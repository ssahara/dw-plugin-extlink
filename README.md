Extend Link Syntax plugin
=========================
Extend DokuWiki Link syntax to open the linked page in a new tab, window

Syntax/Usage
------------


    [[!> ns:page| **bold** title for the link]] 

    {{!> ns:example.pdf| //fancy title of the media file// }}
